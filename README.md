# Smart Career Advisor: AI-Driven Job Recommendations and Skill Gap Analysis

## 📘 Project Overview
In the rapidly evolving job market, individuals face challenges in finding suitable roles and identifying skill gaps.  
This project presents an AI-powered platform that analyzes user resumes, identifies skill gaps, and recommends jobs and learning paths using models like **BERT**, **Sentence-BERT**, and **GPT-5-mini**.

## 🎯 Objectives
- Analyze user resumes using NLP
- Recommend suitable job opportunities
- Identify missing skills and suggest learning resources
- Provide personalized career advice via AI

## ⚙️ Tech Stack (Planned)
| Layer | Technologies |
|--------|---------------|
| Backend | FastAPI, Python, PostgreSQL, Faiss |
| AI Models | BERT, Sentence-BERT, GPT-5-mini |
| Frontend | React, Tailwind CSS |
| Deployment | Docker, Cloud (Render / AWS) |
